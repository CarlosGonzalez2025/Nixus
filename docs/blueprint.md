## Core Features:

- User Authentication: Secure login functionality with role-based access control.
- Dashboard Overview: Display key permit statistics, pending approvals, and quick actions.
- Permit Creation: Step-by-step form for creating and submitting work permits.
- Risk Assessment Tool: Integrate a tool that utilizes generative AI to evaluate risk based on work type and environmental factors. The AI then recommends the appropriate controls for users.
- Permit List and Details: View, filter, and manage existing work permits with detailed information.
- Mobile Responsiveness: Ensure the application is fully responsive and accessible on all devices.

## Style Guidelines:

- Primary color: Deep blue (#fb9404) for a sense of trust and security.
- Background color: Light gray (#F9FAFB) to ensure readability and a clean interface.
- Accent color: Vibrant green (#4ADE80) for positive actions and confirmations.
- Body and headline font: 'Inter' sans-serif for a modern, clean, and readable experience.
- Use consistent and clear icons from Lucide to represent actions and categories.
- Maintain a clean and intuitive layout with clear separation of sections and elements.
- Subtle transitions and animations to enhance user experience.
